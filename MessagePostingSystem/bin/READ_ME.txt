EXECUTION INSTRUCTIONS:
----------------------
Once Extracted. Execute below to run the program.
Come to the directory where this readme is present(src folder)
To compile:
-----------
javac messagePosting/Main.java 



To Run:
-------

Depending whether client or server machine,following commands are used.

java messagePosting.Main server
(or)
java messagePosting.Main client 


The port Number or hostname need not be specified as it already assigned in the program
The hostname is got using InetSocketaddress class
